# Bússola Política Portugal

Uma ferramenta educacional interativa para ajudar os portugueses a compreender a sua posição no espectro político através de análise profunda e imparcial.

## Características

- **100 Perguntas Profundas**: Cobrindo 8 categorias temáticas (Economia, Autoridade, Valores Sociais, Soberania, etc.)
- **4 Dimensões Políticas**: Análise em 4 eixos (Económico, Autoridade, Social, Soberania)
- **23 Partidos Portugueses**: Comparação com partidos posicionados com base em programas políticos
- **Total Privacidade**: Dados armazenados apenas no seu navegador (localStorage)
- **Histórico de Resultados**: Guarde e compare até 10 resultados anteriores
- **Partilha de Resultados**: URL encoding para partilhar resultados facilmente

## Metodologia

### Dimensões Políticas

1. **Eixo Económico**: Esquerda (intervenção estatal) ↔ Direita (livre mercado)
2. **Eixo de Autoridade**: Libertário (liberdades individuais) ↔ Autoritário (ordem e controlo)
3. **Eixo Social**: Progressista (mudança social) ↔ Conservador (tradição)
4. **Eixo de Soberania**: Globalista (integração europeia) ↔ Nacionalista (soberania nacional)

### Sistema de Pontuação

- Cada pergunta tem ponderações específicas para cada dimensão
- Respostas numa escala Likert de 5 pontos (-2 a +2)
- Ponderação de importância opcional (0-5) para refinar resultados
- Cálculo de distância euclidiana 4D para ranking de partidos


Desenvolvido com o auxílio de assistentes de IA generativa, mas com revisão humana cuidadosa para garantir imparcialidade e precisão.

---

**Nota**: Este projeto tem fins puramente educacionais. Os resultados não constituem aconselhamento político e não devem ser interpretados como recomendações de voto.
